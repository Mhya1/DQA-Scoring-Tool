# DQA-Scoring-Tool
Data Quality Assessment (DQA) Scoring Tool for humanitarian and development programmes. Covers 5 dimensions — Completeness, Accuracy, Timeliness, Consistency, and Confidentiality — with 22 weighted indicators, auto-generated recommendations, and a print-ready PDF report. Built with React + Vite. Aligned with USAID, WFP, and PEPFAR DQA standards.
